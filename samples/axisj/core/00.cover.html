$noscript$

AXISJ Core의 구성요소

  • AXConfig.js - 액시스제이 환경설정 변수 모음
  • AXUtil.js - Prototype함수, util함수, Class 패턴 지원
  • AXCore.js - UI Class원형, 공용으로 쓰이는 UI클래스들 (AXMask, AXScroll..)

위 3가지 파일을 합쳐서 AXJ.js 파일을 만들고 다시 Uglify처리하여 AXJ.coremin.js 파일을 만듭니다.
